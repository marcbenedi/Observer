## Installation
Add the JitPack repository to your build file
```gradle
allprojects {
	repositories {
		...
		maven { url 'https://jitpack.io' }
	}
}
```

Add the dependency
```gradle
dependencies {
  implementation 'com.github.marcbenedi:Observer:1.0'
}
```

See more instructions on [Jitpack](https://jitpack.io/#marcbenedi/Observer)
